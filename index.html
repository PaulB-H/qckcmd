Quick Commands

VS Code

! Note
Cursor location determines scope when collapsing a single region or subregion

ALL CAPS = user defined 


Ctrl + `

Show / hide integrated terminal 

Alt + B

Opens current file in browser (with this extension) 

Ctrl + Shift + [

Fold (collapse) region 

Ctrl + Shift + ]

Unfold (uncollapse) region

! Note
For the following commands, keep Ctrl held but press the other keys one after another!


Ctrl + K -> Ctrl + [

Fold (collapse) all subregions 

Ctrl + K -> Ctrl + ]

Unfold (uncollapse) all subregions 

Ctrl + K -> Ctrl + 0

Fold (collapse) all regions 

Ctrl + K -> Ctrl + J

Unfold (uncollapse) all regions

Git

git init

Initializes a LOCAL git repository in the current dir 

git add .

Add all saved files to next commit 

git add FILENAME

Add specific file(s) to next commit
Specify dir as DIR/DIR/FILENAME 

git add -p

Lets you submit chunks even within a certian file.
You can also break up chunks into smaller parts, to a certain limit. 

git commit -m "SUBJECT" -m "DETAILS"

SUBJECT recommend character length = 50, max = 72. DETAILS put anything more. 

git push origin TARGET

Pushes to specified remote, typically "main"

GitHub CLI

gh repo create

Creates a remote repo on GitHub, and initializes it as an active remote to push/pull

Heroku CLI

heroku create appname

Creates a heroku application. You may leave appname blank, and a random name will be used.

SSH

ssh USERNAME@IPADDRESS

Open SSH connection to target IP as USERNAME

Ubuntu

apt get update && apt get upgrade

Download current package information
&& install available upgrades 

ls

List files in current dir 

cd TARGET

Change directory 

mkdir DIRNAME

Create directory 

adduser USERNAME

Create user 

su USERNAME

Switch user 

usermod -aG sudo USERNAME

Grant user sudo permission

Ubuntu UFW - Uncomplicated Firewall

sudo ufw default deny incomming

Deny all incomming connections 

sudo ufw default allow outgoing

Allow all outgoing connections 

sudo ufw deny http

Deny http connections 

sudo ufw allow ssh

Allow SSH connections
ENSURE THIS COMMAND IS RUN BEFORE ENABLING UFW OVER SSH 

sudo ufw enable

Enable UFW
ENSURE "sudo ufw allow ssh" HAS BEEN RUN FIRST WHEN ENABLING UFW OVER SSH 

sudo ufw status verbose

Display detailed UFW status
