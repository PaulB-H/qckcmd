VS Code

! Note
Cursor location determines scope when collapsing a single region or subregion

Ctrl + `

Show / hide integrated terminal 

Alt + B

Opens current file in browser (with this extension) 

Ctrl + Shift + [

Fold (collapse) region 

Ctrl + Shift + ]

Unfold (uncollapse) region

! Note
For the following commands, keep Ctrl held but press the other keys one after another!


Ctrl + K -> Ctrl + [

Fold (collapse) all subregions 

Ctrl + K -> Ctrl + ]

Unfold (uncollapse) all subregions 

Ctrl + K -> Ctrl + 0

Fold (collapse) all regions 

Ctrl + K -> Ctrl + J

Unfold (uncollapse) all regions

Git

git init

Initializes a LOCAL git repository in the current dir 

git add .

Add all saved files to next commit 

git add FILENAME

Add specific file(s) to next commit
Specify dir as DIR/DIR/FILENAME 

git add -p
 -- -- -- --
(1/10) Stage this hunk [y,n,q,a,d,j,J,g,/,s,e,?]? ?
y - stage this hunk
n - do not stage this hunk
q - quit; do not stage this hunk or any of the remaining ones
a - stage this hunk and all later hunks in the file
d - do not stage this hunk or any of the later hunks in the file
g - select a hunk to go to
/ - search for a hunk matching the given regex
j - leave this hunk undecided, see next undecided hunk
J - leave this hunk undecided, see next hunk
s - split the current hunk into smaller hunks
e - manually edit the current hunk
? - print help

Add specific hunks from within a file(s) for commit.
You can also split the hunks into smaller parts. 

git commit -m "SUBJECT" -m "DETAILS"

SUBJECT recommend character length = 50, max = 72. DETAILS put anything more. 

git push origin TARGET

Pushes to specified remote, typically "main"

Ubuntu UFW
Uncomplicated Firewall

sudo ufw default deny incoming

Deny all incoming connections 

sudo ufw default allow outgoing

Allow all outgoing connections 

sudo ufw deny http

Deny http connections 

sudo ufw allow ssh

Allow SSH connections
ENSURE THIS COMMAND IS RUN BEFORE ENABLING UFW OVER SSH 

sudo ufw enable

Enable UFW
ENSURE "sudo ufw allow ssh" HAS BEEN RUN FIRST WHEN ENABLING UFW OVER SSH 

sudo ufw status verbose

Display detailed UFW status
