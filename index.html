 ᐯ Emmet Abbreviations

The following settings must be enabled in VS Code settings.json file in order for emmet to work correctly in JSX (mainly for React) 

"emmet.includeLanguages": {
  "\"javascript\"": "\"javascriptreact\""
},
"emmet.triggerExpansionOnTab": true,
"emmet.showSuggestionsAsSnippets": true,
"files.associations": {
  "*.js": "javascriptreact"
},
